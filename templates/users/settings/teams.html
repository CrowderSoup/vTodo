Skylight has no developer login, so vtodo needs a one-time refresh token pulled from your own browser session instead of your password. To get it:

  1. In Chrome/Firefox, open DevTools (F12 or Cmd+Opt+I) and go to the Network tab.
  2. Go to app.ourskylight.com and log in normally (or refresh the page if already logged in).
  3. In the Network tab's filter box, type oauth/token.
  4. Click the token request that appears, open its Response (or Preview) panel, and copy the value of refresh_token — a long string, not access_token.
  5. Paste that value above. It's used once to connect, then vtodo rotates it automatically on its own after that — you won't need to repeat this unless the connection needs reconnecting.

The frame ID isn't exposed anywhere in the app either — while you're in that same Network tab, look for a request URL containing /api/frames/{frame_id}/... and copy the id from there.
